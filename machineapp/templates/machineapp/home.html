About the Project

This is a Machine failure Prediction System designed to analyze machine health using real sensor data. Users can upload a CSV file containing machine readings, and the system automatically processes the data,trains an Ml model, and displays meaningful insights.

Temperature - Measures heat generated during machine operation.
Vibration - Indicates mechanical imbalance or friction.
Pressure - Shows internal system pressure levels.
Failure (0/1) - Whether the machine failed or not.
